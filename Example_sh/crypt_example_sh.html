crypt_example_sh

This repository contains a script file for Bash, that describes step-by-step generation of the Metahash address with OpenSSL CLI Tool. For more details about Metahash address generation, please read the article.

Run the script

./crypt_example.sh function [parameter]

List of functions:
    generate - generate MetaHash address with OpenSSL CLI Tool
    usage - display help info

Fore more details see https://github.com/metahashorg/crypt_example_sh
